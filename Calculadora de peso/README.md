# Calculadora de Alimentación Canina

Proyecto minimalista: calculadora que sugiere cantidad de comida, peso ideal y variantes recomendadas por raza y edad. 

Archivos principales:
- `index.html` — interfaz
- `styles.css` — estilos
- `script.js` — datos y lógica

Cómo usar:
1. Abrir `index.html` en el navegador.
2. Seleccionar raza y edad, pulsar "Calcular recomendaciones".
